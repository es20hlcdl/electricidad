# Encuesta de Electricidad

Aplicación web interactiva para visualizar datos geoespaciales relacionados con encuestas de electricidad en diferentes distritos.
Puedes visitarlo a través del siguiente link: https://es20hlcdl.github.io/electricidad/

## 🚀 Características

- Mapa interactivo con Leaflet
- Visualización de distritos y vías
- Datos geoespaciales en formato GeoJSON
- Interfaz responsiva y amigable

## 📁 Estructura del Proyecto

```
.
├── index.html              # Página principal
├── css/                    # Estilos (Leaflet, qgis2web, FontAwesome)
├── js/                     # Librerías y scripts (Leaflet, mapas)
├── data/                   # Datos GeoJSON y configuración
└── images/                 # Imágenes y assets
```

## 🗺️ Datos Disponibles

- Distritos
- Caminos y vías
- Límites administrativos (Concepción, Cuatro Cañadas, San Antonio Lomerío, San Ignacio, San José Chiquitos, San Miguel, San Rafael)

## 📖 Uso

1. Abre `index.html` en tu navegador web
2. Interactúa con el mapa para explorar los datos
3. Usa los controles del mapa para zoom y navegación

## 🛠️ Tecnologías

- Leaflet.js - Mapas interactivos
- GeoJSON - Formato de datos geoespaciales
- HTML5 / CSS3 / JavaScript - Frontend
